dynatrace-web-reporter

(early stage)

  • Leverages dynaTrace REST interfaces for reporting and server management.

  • Provides a web frontend to SaaS dynaTrace instance.

  • Uses smartgwt for windowing and datasource management.

  • Uses GWT HighCharts for charting.
